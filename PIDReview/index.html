ArduPilot PID Review Tool

This tool takes a .bin log with RATE or PID messages and shows the time and frequency content of the rate controller target, response and output. To record the full set of PID components the PID bit of the LOG_BITMASK parameter must be set before flying. The RATE log message is enabled by default and can also be used by this tool.
