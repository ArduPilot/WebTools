Welcome to ArduPilot Web Tools

A number of web-based tools for ArduPilot log and parameter review. These tools operate on the client side, ensuring your data stays private. Many tools support the "Open In" feature to transfer logs between tools seamlessly.

Log Finder

Load and sort directories of logs. Tracks parameter changes and provides easy log transfer between tools.

UAV Log Viewer

General-purpose log review with 3D flight visualization.

Hardware Report

Overview of connected hardware, sensor health, and firmware details.

MAGFit

Calibrate compass from flight logs using world magnetic models.

Filter Review

Analyze gyro noise and filter configurations without re-flying.

PID Review

Review PID tunes in the frequency domain with step response estimates.

Filter Tool

Visualize filter setups and analyze attenuation and phase lag.

Work in Progress Tools

Explore experimental tools under development.
